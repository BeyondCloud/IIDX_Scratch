This script will push down the space botton when you move the mouse,release when the mouse stop.
You should install AutoHotKey before using the script
How to use:
1.Double click the script,if nothing going wrong you will see a green "H" icon appear at the botton right of screen
2.Press left ctrl key to run the script,your mouse wont be able to move.Now the mouse move will act as space
3.press L-ctrl again to stop the script
4.press F8 to close the script(you can change any key you want at the end of script,to edit script,right click the
script and choose edit will do the job )
